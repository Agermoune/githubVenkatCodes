Variable Types¶

  • Varibles are automically assigned by python to have certain types like in excel or SQL

Text Types:

  • str = string

Numerical Types:

  • int = integer (1,2,3)

  • float = floatation Point Number (1.5, 1.0, 5.3)

Sequence or Grouping Types:

  • list = list of items [1,3,4,'hi']

  • tuple = unchanging list of items (1,2,3,'hi')

  • dict = dictionary or hashmap, key-value pairs {1:'first',2:'second'} (more on this later)

Boolean or T/F

  • bool = boolean True/False

find the type of any variable using type()

  • The user can change the types using built in data type functions

y = 6.0
z = 'hi'

Dictionaries¶

  • Serve as a key-value pair collection
  • A dictionary is a collection which is unordered, changeable and indexed. In Python dictionaries are written with curly brackets, and they have keys and values.
In [9]:
thisdict = {
  "brand": "Ford",
  "model": "Mustang",
  "year": 1964
}
print(thisdict)

{'brand': 'Ford', 'model': 'Mustang', 'year': 1964}

Can access items by passing the key through the dict variable

In [10]:
x = thisdict["model"]
print(x)

Mustang

In order to change values you can declare it similar to variable declaration but pass in the key and value you want

In [11]:
thisdict["year"] = 2018
In [12]:
thisdict
Out[12]:
{'brand': 'Ford', 'model': 'Mustang', 'year': 2018}

String Manipulation¶

String literals in python are surrounded by either single quotation marks, or double quotation marks.

Like many other popular programming languages, strings in Python are arrays of bytes representing unicode characters.

You can return a range of characters by using the slice syntax.

Similar to Left() or Right() in excel

In [19]:
b = "Hello, World!"
print(b[-5:-2])
print(b[2:5])

orl
llo
